# NODE 1 — Smart Power Socket

NODE 1 is a smart power socket offering remote control, smart scheduling, powercut detection, and weather alerts. Built for unstable power environments, it ensures reliable operation, continuous monitoring, and intelligent appliance management.

## Key features
- **REMOTE CONTROL** — Switch appliances On/Off from anywhere.
- **SMART SCHEDULING** — Automate power usage with ease.
- **POWERCUT DETECTION** — Protects devices from unstable supply.
- **WEATHER ALERTS** — Stay informed about local conditions.

## Repository structure
- `Hardware/` — Schematics, PCB, Gerbers
- `Firmware/` — ESP32 source code
- `Enclosure/` — 3D model (STL) & renders
- `Docs/` — Reports, Diagrams
- `Images/` — Photos, Presentations
